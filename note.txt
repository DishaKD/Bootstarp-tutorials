Boot strap Containers
  - Important an Basic layout element in Bootstrap
  - There 2 type of Containers
     - <div class="container"></div> //fixed width
     - <div class="container-fluid"></div> // 100% of the width of the screen size

  - Fluid container 
    - width is always 100%

  - Fixed container
    - Dimensions of fixed container
      - Extra Small < 576px - 100% (Max width)
      - Small >= 576px - 540px (Max width)
      - Medium >= 768px - 720px (Max width)
      - Large >= 992px - 960px (Max width)
      - Extra Large >= 1200px - 1140px (Max width)
  
  - Why use fixed layout at all ?
  - static pixels
